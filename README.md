# waku-waku 
和阿尼亚一起哇酷哇酷吧！

## 简介
waku-waku 是一个通用管理平台(UMP, Universal Management Platform). 内置多种工具并同时支持 CLI 和 GUI 模式.

## 模块列表
- waku-core: 哇酷核心
- waku-cli: 哇酷命令行
- waku-gui: 哇酷图形界面
- waku-utils: 哇酷工具链

## 支持工具列表
- QuickBMS with QuickBMS scripts.
- vgmstream.
- dotnet-script.
- ffmpeg.
- [AssetStudio][AssetStudio].
- [map-viewer][map-viewer].
- [it-is-a-exporter][it-is-a-exporter].
- hex-viewer
- skia-image-render
